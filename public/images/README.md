# Image Assets for NotATree Story

Add your photos here with these recommended names:

## Story Panels

1. **sketch.jpg** - First concept drawing/sketch
2. **blueprint.jpg** - Technical drawings/refinement phase  
3. **installation.jpg** - The built structure (7-meter installation)
4. **vision.jpg** - Optional: Vision/future concept image
5. **impact.jpg** - Optional: Installation in use/impact photo
6. **data.jpg** - Optional: Data visualization/sensors in action

## Recommended Specs

- Format: JPG or PNG
- Size: 1200-1600px width recommended
- Aspect ratio: 4:3 or 16:9 works best
- Optimized for web (under 500KB each)

## How to Add

1. Drop your images into this folder
2. The NotATree component is already set up to use them
3. Images will automatically optimize via Next.js Image component
